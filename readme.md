# Getting Started

Berikut cara untuk menjalankan program pada git local.

# Installation

cloning repo dengan cara

```sh
git clone git@github.com:agungcahya122/venturo.git
```

Install semua package.json

```sh
npm install
```

  <br />

jalankan aplikasi

```sh
npm run dev
```
